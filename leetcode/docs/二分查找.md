# 二分查找
伪代码
```
while (start <= end) {
    int mid = (end - start) / 2 + start;
    if (mid < target) {
        start = mid + 1;
    } else if (mid > target) {
        end = mid - 1;
    } else {
        return mid; 
    }
}
```  
要点如下
- while条件中，start要<=end，是因为当元素只有1个时，也可以进行查找，不会越界
- 左右指针需要mid +/- 1，以防止在只有两个元素时指针静止，产生死循环
- 使用 (end - start) / 2 + start 求中点可以防止数据溢出
- 以5、6为例，mid分别为2、3，mid在元素有奇数个时恰为中点，为偶数个时为中点左侧
